# nextauth-demo

Learning NextAuth.js with OAuth authentication with Prisma adapter

### Special feature

- Added new field `role` for `User` model

### `.env`

Structure your `.env` file like so

```sh
NEXTAUTH_SECRET=

GITHUB_ID=
GITHUB_SECRET=

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=

FACEBOOK_CLIENT_ID=
FACEBOOK_CLIENT_SECRET=

DATABASE_URL=
```
